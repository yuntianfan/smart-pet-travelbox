# ADR-0001: 租赁优先商业模式

**状态:** 已确定  
**日期:** 2026-06-01

## 背景

项目初期目标是打造一款智能宠物旅行箱，让宠物主在托运过程中能实时了解宠物状态。商业模式需要在零售、租赁、共享三种模式中选定切入点。

## 决策

**MVP 采用租赁模式，平台拥有箱子资产，向宠物主出租。**

宠物主通过小程序预约箱子，前往宠物店分舵取箱，行程结束后归还分舵。镖师（司机）在 MVP 阶段不强制入驻，宠物主自行安排托运服务。

## 理由

- **低频需求**：宠物托运对宠物主而言是低频场景，零售模式下的设备投资门槛不利于初期推广
- **资产可控**：平台拥有箱子，统一管理设备生命周期（维护、充电、消毒、回收）
- **信任建立**：平台持有箱子才能真正承担透明度和告警责任
- **后期扩展**：当平台积累足够订单流量后，可引导入驻镖师长租或购买箱子，形成双边市场

## 不采用其他方案的原因

- **零售**：低频场景下宠物主购买意愿低，不利于初期市场验证
- **共享**：多用户共用同一箱子涉及复杂的卫生、损坏、归还协调问题，MVP 阶段过于复杂
- **强制镖师入驻**：镖师入驻会增加平台运营复杂度，M平台初期专注箱子体验而非服务质量

## 影响

- 平台承担箱子采购和维护成本
- 分舵作为箱子物理流转节点，需支持取箱/还箱扫码操作
- 计费按实际使用天数结算（非固定租期）
